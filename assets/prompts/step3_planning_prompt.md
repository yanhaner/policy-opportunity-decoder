# 步骤3：项目策划提示词

> 用于指导AI完成政策文件分析的第三步——项目策划与产品定义

---

## 任务目标

将机会和需求打包成具体的、可销售的服务产品，形成《项目策划概要》。

---

## 分析框架

### 1. 项目命名公式

```
[地域/IP] + [政策关键词] + [创新场景] + [价值后缀]

示例：
• 宜宾三江新区 "人工智能+" 产业融合先导区 总体规划
• 成都东部新区 低碳智慧 城市更新 示范工程策划
• 川南经济区 乡村振兴 EOD模式 投融资方案
```

### 2. 服务积木组合策略

#### 业务类型分类
| 类型 | 代码 | 服务内容 | 适用场景 |
|-----|------|---------|---------|
| **研究类** | A | 政策研究、产业研究、专题研究、项目概念策划 | 前期策划阶段 |
| **规划类** | P | 总体规划、专项规划、详细规划、空间规划、产业规划、城市更新规划、全域土地综合整治实施方案、区域规划、发展规划、文旅融合规划 | 规划设计阶段 |
| **咨询类** | C | 战略咨询、管理咨询、可研报告、评估报告、技术咨询、产业导入、全过程工程咨询、项目策划 | 全程支持 |
| **设计类** | E | 建筑设计、景观设计、工程设计、EPC | 落地建设阶段 |
| **数字化** | D | 智慧城市、数字化平台设计、数据要素应用、场景规划 | 数字赋能与业务协同 |
| **投融资** | F | 专项债谋划、EOD策划、REITs、特许经营、融资可行性研究 | 资金谋划与项目入库 |
| **运营类** | O | 招商引资、产业培育、物业管理、活动策划、运营策划、内容运营 | 后期长效运营阶段 |

#### 服务包设计
```
基础包（必选）：核心服务模块
    ↓
升级包（可选）：增值服务模块
    ↓
定制包（特选）：个性化服务模块
```

### 3. 价值主张设计公式（V = I + C + T + S）

| 维度 | 含义 | 评分标准（1-5分） |
|-----|------|-----------------|
| **I** - Impact | 战略影响力 | 能否写入政府工作报告？能否成为区域标杆？ |
| **C** - Connection | 诉求连接度 | 是否直接解决核心决策者的KPI或痛点？ |
| **T** - Timing | 时间匹配度 | 是否契合政策窗口期和资金申报节奏？ |
| **S** - Synergy | 业务协同度 | 能否带动机构内部3个及以上业务部门参与？ |

> **评分规则**：单项1-5分，总分≥12分视为**高价值机会**

## 分析要点

### DO（应该做）

- [ ] 使用命名公式，确保项目名称专业规范
- [ ] 设计模块化服务包，便于客户选择
- [ ] 客观进行V公式评分，不夸大
- [ ] 明确服务内容和交付成果
- [ ] 合理估算工作周期和费用
- [ ] 考虑竞争对手的差异化策略

### DON'T（不应该做）

- [ ] 不要设计过于复杂的服务包
- [ ] 不要低估工作量和周期
- [ ] 不要忽视客户的支付能力
- [ ] 不要抄袭竞争对手的方案
- [ ] 不要承诺无法实现的目标

## 项目评估矩阵

### 机会优先级评估

```
                    高政策确定性
                         │
         ┌───────────────┼───────────────┐
         │   第一象限    │   第二象限    │
         │   立即行动    │   重点跟进    │
         │  高确定性+高价值│  高确定性+高价值│
低市场热度├───────────────┼───────────────┤高市场热度
         │   第三象限    │   第四象限    │
         │   持续关注    │   快速验证    │
         │  低确定性+低价值│  低确定性+高价值│
         └───────────────┼───────────────┘
                         │
                    低政策确定性
```

### 项目可行性评估

| 评估维度 | 评估内容 | 评分 |
|---------|---------|------|
| **技术可行性** | 我们有能力完成吗？ | [1-5分] |
| **商务可行性** | 我们能拿到项目吗？ | [1-5分] |
| **经济可行性** | 项目有合理利润吗？ | [1-5分] |
| **风险可控性** | 主要风险可控吗？ | [1-5分] |

## 示例

### 背景

政策机会：产业园区数字化转型  
目标客户：宜宾三江新区管委会  
客户痛点：产业定位模糊、招商竞争力弱、运营经验不足

### 输出

**项目命名**：宜宾三江新区 "智汇三江" 数字经济产业园 总体规划与招商运营方案

**V公式评分**：

| 维度 | 评分 | 说明 |
|-----|------|------|
| **I - Impact** | 5分 | 可写入市政府工作报告，打造省级数字经济标杆 |
| **C - Connection** | 5分 | 直接解决新区管委会产业培育KPI |
| **T - Timing** | 4分 | 契合数字经济发展政策窗口期 |
| **S - Synergy** | 5分 | 可带动产业规划、空间规划、招商运营、智慧设计4个部门 |
| **总分** | **19分** | ✅ 高价值机会 |

**服务模块设计**：

基础包：
1. 产业现状调研与诊断
2. 数字经济产业定位与目标体系
3. 空间布局与功能分区规划
4. 产业链设计与招商方向
5. 实施路径与保障措施

升级包：
1. 智慧园区建设方案（+30万）
2. 重点招商企业靶向清单（+20万）
3. 园区运营管理模式设计（+25万）
