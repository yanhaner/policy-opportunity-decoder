# 政策机会解码器

> 🎯 **技能定位**：政策翻译器 × 价值包装师  
> 将宏观政策系统性地解码为可落地、高价值的工程咨询与设计服务项目机会

---

## 📋 技能总览

本技能专为区域发展规划设计咨询机构打造，提供一套完整的**政策文件项目机会解码与包装方法论**。通过四步结构化流程，帮助策略专家将抽象的政策文本转化为具体的商业机会和专业化方案。

### 核心能力

| 能力模块 | 功能描述 |
|---------|---------|
| 🔍 **政策解码** | 像"手术刀"一样拆解政策，提取关键信号和项目线索 |
| 💡 **需求挖掘** | 穿透政策表面，洞察执行者的显性与隐性需求 |
| 📦 **项目策划** | 将机会和需求打包成可销售的服务产品 |
| 🎨 **方案包装** | 将专业方案转化为打动决策者的商业故事 |

## 🧠 核心方法论

### 1. 政策-资金-项目转化漏斗

```
┌────────────────────────────────────────────────────────┐
│  政策条款（宏观导向）                                    │
│  例："推动数字经济与实体经济深度融合"                   │
└────────────────────┬───────────────────────────────────┘
                     ▼
┌────────────────────────────────────────────────────────┐
│  资金支持（中观渠道）                                    │
│  例：数字经济发展专项资金、新基建专项债                   │
└────────────────────┬───────────────────────────────────┘
                     ▼
┌────────────────────────────────────────────────────────┐
│  项目机会（微观落地）                                    │
│  例：产业园区数字化改造规划、工业互联网平台建设咨询       │
└────────────────────────────────────────────────────────┘
```

### 2. 需求冰山模型

```
        /\
       /  \\      【显性需求】政策明确要求完成的任务
      /____\\       例：建设高标准农田100万亩
     /      \\
    /        \\    【隐性需求】执行中的真实痛点
   /__________\\    例：资金缺口大、技术标准缺失、考核压力大
  /            \\
 /              \\ 【深层需求】政治/经济/社会综合诉求
/________________\\  例：打造政绩亮点、拉动GDP、改善民生
```

### 3. 价值主张设计公式（V = I + C + T + S）

| 维度 | 含义 | 评分标准（1-5分） |
|-----|------|-----------------|
| **I** - Impact | 战略影响力 | 能否写入政府工作报告？能否成为区域标杆？ |
| **C** - Connection | 诉求连接度 | 是否直接解决核心决策者的KPI或痛点？ |
| **T** - Timing | 时间匹配度 | 是否契合政策窗口期和资金申报节奏？ |
| **S** - Synergy | 业务协同度 | 能否带动3个及以上业务部门参与？ |

> **评分规则**：单项1-5分，总分≥12分视为**高价值机会**

### 4. 项目命名公式

```
[地域/IP] + [政策关键词] + [创新场景] + [价值后缀]

示例：
├─ 宜宾三江新区 "人工智能+" 产业融合先导区 总体规划
├─ 成都东部新区 低碳智慧 城市更新 示范工程策划
└─ 川南经济区 乡村振兴 EOD模式 投融资方案
```
